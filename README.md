# Robot Visualization System

## Описание проекта

Разработан прототип системы визуализации и мониторинга мобильного робота в реальном времени.

Система принимает телеметрические данные с робота по Wi-Fi (в проекте используется эмуляция), обрабатывает их на компьютере и отображает:

- положение робота
- траекторию движения
- карту окружающего пространства
- данные лидара (облако точек)
- текущие параметры состояния

---

## Цель

Создать систему, позволяющую оператору в реальном времени наблюдать за состоянием робота и окружающей средой без нагрузки на бортовой компьютер.

---

## Технологический стек

- Python
- NumPy
- Matplotlib
- Requests (HTTP API)
- Flask (эмуляция робота)

---

## Архитектура системы

Fake Robot (API)
        ↓
Receiver (приём данных)
        ↓
Telemetry Filter (фильтрация)
        ↓
Lidar Processing (обработка точек)
        ↓
Visualization (карта и интерфейс)

## Запуск проекта

1. Установить зависимости:
pip install -r requirements.txt

2. Запустить эмулятор:
python fake_robot.py

3. Запустить визуализацию:
python wow_map.py

---

## Управление

L — лидар  
T — траектория  
M — метрики  
C — очистка карты  
Q — выход
